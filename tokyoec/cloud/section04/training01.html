1.EFKの環境構築

  1. 最新版のDockerインストール
    curl -fsSL https://get.docker.com/ | sh
  2. Docker Composeのインストール
    curl -L https://github.com/docker/compose/releases/download/1.9.0/docker-compose-`uname -s`-`uname -m` > /usr/local/bin/docker-compose
  3. ダウンロードしたバイナリに対して実行権限を追加
    chmod +x /usr/local/bin/docker-compose
  4. インストール確認
    docker-compose -v
  5. サンプルアプリケーションのクローン
    git clone https://github.com/efkbook/blog-sample
  6. サンプルアプリケーションの起動
    cd blog-sample
    docker-compose up -d
  7. 環境起動
    docker-compose ps
  8. マイグレーションの実行
    docker-compose exec go make migrate/up
  9. Elasticsearchの動作確認
    curl グローバルIPアドレス:9200
