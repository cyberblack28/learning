EC-CUBEのインストール

1.EC-CUBEのインストール

  1. EC-CUBEのインストール
    cd /var
    wget http://downloads.ec-cube.net/src/eccube-3.0.16.zip
  2. unzipコマンドのインストールと解凍
    yum install -y unzip
    unzip eccube-3.0.16.zip
  3. 対象ファイルをドキュメントルートに上書き
    cd eccube-3.0.16
    mv * ../www/
    mv: overwrite ‘../www/html’? y ←yと入力してEnterキー
    mv .htaccess ../www/
  4. SELinuxの無効化
    setenforce 0
    getenforce
  5. EC-CUBEの必要モジュールのインストール
    yum install -y epel-release
    yum install -y php-mcrypt php-pecl-apcu
  6. ドキュメントルートの所有者の設定
    cd /var/www
    chown -R apache:apache *
  7. httpd.confの設定
    cp -p /etc/httpd/conf/httpd.conf /etc/httpd/conf/httpd.conf.org
    vi /etc/httpd/conf/httpd.conf
