PHPのインストール

1.PHPのインストール

  1. phpバージョン7系のインストール
    yum install -y epel-release
    yum -y install http://rpms.famillecollet.com/enterprise/remi-release-7.rpm
    yum install -y --enablerepo=remi,remi-php70 php php-devel php-mbstring php-pdo php-mysql php-gd php-mcrypt libmcrypt
  2. phpのバージョン確認
    php -v
  3. php.iniの設定
    vim /etc/php.ini
    ----------------------------------
    ↓「Shift + G」で最終行に移動して以下追記↓
    extension=msql.so
    ----------------------------------
    Esc + :wq
  4. httpdの再起動
    systemctl restart httpd
    systemctl status httpd
