1.Dockerのインストール

rancher-server作業

  1. Dockerのインストール
    curl https://releases.rancher.com/install-docker/17.03.2.sh | sh
  2. Rancherのインストール
    docker run -d --restart=unless-stopped -p 80:80 -p 443:443 rancher/rancher
  3. Rancher Serverにアクセス
    ブラウザを起動して、rancher-serverの「https://グローバルIPアドレス/」にアクセス。
