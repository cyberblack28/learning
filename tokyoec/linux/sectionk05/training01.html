PHP環境の構築テキスト

1.phpのインストール

CentOS7_WORDPRESSの仮想マシンで実行すること

  1. phpのインストール
    yum install -y epel-release
    yum -y install http://rpms.famillecollet.com/enterprise/remi-release-7.rpm
    yum install -y --enablerepo=remi,remi-php72 php php-devel php-mbstring php-pdo php-mysql php-gd php-mcrypt libmcrypt
  2. phpのバージョン確認
    php -v
  3. phpの設定情報をブラウザに表示するためのコードを記述
    cd /var/www/html
    vim phpinfo.php
    <?php
phpinfo();
    Esc + :wq
    systemctl restart httpd
  4. ブラウザを起動して、設定情報を確認
    http://192.168.56.28/phpinfo.php
