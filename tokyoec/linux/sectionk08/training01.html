8.FTPサーバの構築

FTPサーバの構築テキスト

1.vsftpdのインストール

CentOS7_WORDPRESSの仮想マシンで実行すること

  1. FTP専用ユーザの作成
    useradd ftpuser
    passwd ftpuser
    ユーザー ftpuser のパスワードを変更。
    新しいパスワード:ftp123@
    よくないパスワード: このパスワードには一部に何らかの形でユーザー名が含まれています。
    新しいパスワードを再入力してください: ftp123@
    passwd: すべての認証トークンが正しく更新できました。
  2. 新ドキュメントルートの作成
    mkdir -p /home/ftpuser/htdocs/public_html
    chown -R ftpuser.ftpuser /home/ftpuser/htdocs
    chmod 755 -R /home/ftpuser
  3. vsftpdをインストール
    yum install -y vsftpd
  4. vsftpd設定ファイルの変更
    cd /etc/vsftpd
    cp -p vsftpd.conf vsftpd.org
    vim vsftpd.conf
    以下変更
anonymous_enable=YES ⇒ anonymous_enable=NO
#ascii_upload_enable=YES ⇒ ascii_upload_enable=YES
#ascii_download_enable=YES ⇒ ascii_download_enable=YES
#chroot_local_user=YES ⇒ chroot_local_user=YES
#chroot_list_enable=YES ⇒ chroot_list_enable=YES
#chroot_list_file=/etc/vsftpd/chroot_list ⇒ chroot_list_file=/etc/vsftpd/chroot_list
#ls_recurse_enable=YES ⇒ ls_recurse_enable=YES
最終行に以下追記
userlist_deny=NO
userlist_file=/etc/vsftpd/ftpuser_list
user_config_dir=/etc/vsftpd/vsftpd_user_conf
allow_writeable_chroot=YES
    Esc + :wq
  5. chroot_listファイルの作成
    touch chroot_list
  6. vim ftpuser_list
    ftpuser
    Esc + :wq
  7. 初期ディレクトリの設定
    mkdir vsftpd_user_conf
    vim vsftpd_user_conf/ftpuser
    local_root=/home/ftpuser/htdocs
    Esc + :wq
  8. SELinuxの設定
    setsebool -P allow_ftpd_full_access on
  9. vsftpdサービスの自動起動設定
    systemctl enable vsftpd
    systemctl list-unit-files --type=service | grep vsftpd
  10. vsftpdの起動
    systemctl start vsftpd
    systemctl status vsftpd
  11. 外部からアクセスできるようにfirewalldでftpを解放
    firewall-cmd --add-service=ftp --zone=public --permanent
  12. 設定を反映
    firewall-cmd --reload
  13. httpdの設定
    vim /etc/httpd/conf/httpd.conf 
    DocumentRoot "/var/www/html"
⇓ 下記に変更
DocumentRoot "/home/ftpuser/htdocs/public_html"
<Directory "/var/www/html">
⇓ 下記に変更
<Directory "/home/ftpuser/htdocs/public_html">
  14. SELinuxの設定
    semanage fcontext -a -t httpd_sys_content_t '/home/ftpuser/htdocs/public_html(/.*)?'
    restorecon -Rv /home/ftpuser/htdocs/public_html
  15. httpdの再起動
    systemctl restart httpd
