6.MariaDBの構築テキスト

MariaDBの構築テキスト

1.mariadbのインストール

CentOS7_WORDPRESSの仮想マシンで実行すること

  1. mariadbのインストール
    yum install -y mariadb mariadb-server wget
  2. mariadbの自動起動設定
    systemctl enable mariadb
  3. mariadbの起動
    systemctl start mariadb
    systemctl status mariadb
  4. mariadbのrootユーザのパスワード設定
    mysql -u root
    MariaDB [(none)]> update mysql.user set password=password('mariadb123') where user = 'root';
    MariaDB [(none)]> FLUSH PRIVILEGES;
    MariaDB [(none)]> exit
  5. WordPress用のDB作成
    mysql -u root -p
    Enter password:mariadb123
    MariaDB [(none)]> CREATE DATABASE wordpress;
    MariaDB [(none)]> GRANT ALL PRIVILEGES ON wordpress.*TO 'wordpress'@'localhost' IDENTIFIED BY 'wppass';
    MariaDB [(none)]> FLUSH PRIVILEGES;
    MariaDB [(none)]> exit

2.WordPressのインストール

CentOS7_WORDPRESSの仮想マシンで実行すること

  1. WordPressのインストール
    wget https://ja.wordpress.org/latest-ja.tar.gz
  2. tarコマンドで展開
    tar xvzf latest-ja.tar.gz
  3. WordPressデータをドキュメントルートに移動
    mv wordpress /var/www/html
  4. semanageコマンドのインストール
    yum install policycoreutils-python
  5. ディレクトリにSELinuxのコンテキストを設定
    semanage fcontext -a -t httpd_sys_content_t '/var/www/html(/.*)?'
    restorecon -Rv /var/www/html
  6. ブラウザでhttp://192.168.56.28/wordpress/にアクセス

3.wp-config.phpファイルの作成

CentOS7_WORDPRESSの仮想マシンで実行すること

  1. wp-config.phpファイルの作成
    cd /var/www/html/wordpress
    vim wp-config.php
    右クリックでコピーした内容をペーストします。
    Esc + :wq
