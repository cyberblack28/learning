東京電子専門学校 Linux 演習課題

6.ネットワーク管理

演習2 DNS関連のコマンド

1.whoisコマンド

  1. カレントディレクトリをホームディレクトリに変更してください

    cd
    pwd

  2. rootユーザに変更してください。

    su -

  3. yumコマンドを利用して、whoisコマンドをインストールしてください。

    yum install -y whois
    rpm -qa | grep whois

  4. whoisコマンドを利用して、東京電子専門学校のドメイン情報（tokyo-ec.ac.jp）を表示してみましょう。自分でも気になるドメインがあれば、whoisで調べてみましょう。

    whois tokyo-ec.ac.jp

  5. studentユーザに戻りましょう。

    exit

2.nslookupコマンド

  1. カレントディレクトリをホームディレクトリに変更してください

    cd
    pwd

  2. nslookupコマンドを利用して、tokyo-ec.ac.jpのグローバルIPアドレスを調べてください。自分でも気になるドメインがあれば、whoisで調べてみましょう。

    nslookup tokyo-ec.ac.jp

  3. nslookupコマンドの対話形式で、tokyo-ec.ac.jpのグローバルIPアドレスを調べてください。

    nslookup [Enterキー]
    ＞tokyo-ec.ac.jp
    ＞exit

3.digコマンド

  1. カレントディレクトリをホームディレクトリに変更してください

    cd
    pwd

  2. digコマンドを利用して、tokyo-ec.ac.jpのaレコード情報を調べてください。

    dig tokyo-ec.ac.jp a

  3. digコマンドを利用して、tokyo-ec.ac.jpのmxレコード情報を調べてください。

    dig tokyo-ec.ac.jp mx

  4. digコマンドを利用して、tokyo-ec.ac.jpのnsレコード情報を調べてください。

    dig tokyo-ec.ac.jp ns

  5. digコマンドを利用して、tokyo-ec.ac.jpのsoaレコード情報を調べてください。

    dig tokyo-ec.ac.jp soa

  6. digコマンドを利用して、tokyo-ec.ac.jpの全情報を調べてください。

    dig tokyo-ec.ac.jp any

4.hostコマンド

  1. カレントディレクトリをホームディレクトリに変更してください

    cd
    pwd

  2. hostコマンドを利用して、tokyo-ec.ac.jpのグローバルIPアドレスを調べてください。

    host tokyo-ec.ac.jp

  3. hostコマンドを利用して、tokyo-ec.ac.jpのaレコード情報を調べてください。

    host -t a tokyo-ec.ac.jp

  4. hostコマンドを利用して、tokyo-ec.ac.jpのmxレコード情報を調べてください。

    host -t mx tokyo-ec.ac.jp

  5. hostコマンドを利用して、tokyo-ec.ac.jpのnsレコード情報を調べてください。

    host -t ns tokyo-ec.ac.jp

  6. hostコマンドを利用して、tokyo-ec.ac.jpのsoaレコード情報を調べてください。

    host -t soa tokyo-ec.ac.jp

  7. hostコマンドを利用して、103.4.43.184のptrレコード情報を調べてください。

    host -t ptr 103.4.43.184

  8. hostコマンドを利用して、google.co.jpのIPv6のアドレスを調べてください。

    host -6 google.co.jp
