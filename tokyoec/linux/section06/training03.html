6.ネットワーク管理

演習3 NetworkManager

1.nmcliコマンド～インターフェイス編～

  1. カレントディレクトリをホームディレクトリに変更してください

    cd
    pwd

  2. rootユーザに変更してください。

    su -

  3. nmcliコマンドを利用して、enp0s3の現在設定されているIPアドレスを表示してください。「grep IP4」を利用してください。

    nmcli connection show enp0s3 | grep IP4 または nmcli c show enp0s3 | grep IP4

  4. nmcliコマンドを利用して、手動設定に切り替えてenp0s3のIPアドレスを「10.0.2.0/24」に変更してください。

    nmcli connection modify enp0s3 ipv4.method manual ipv4.address 10.0.2.0/24 または nmcli c m enp0s3 ipv4.method manual ipv4.address 10.0.2.0/24
    nmcli connection modify enp0s3 ipv4.address 10.0.2.0/24 または nmcli c m enp0s3 ipv4.address 10.0.2.0/24

  5. 手動設定に切り替えたものを自動構成（DHCP）に戻してください。

    nmcli connection modify enp0s3 ipv4.method auto または nmcli c m enp0s3 ipv4.method auto
    nmcli connection modify enp0s3 -ipv4.address 10.0.2.0/24 または nmcli c m enp0s3 -ipv4.address 10.0.2.0/24

  6. 設定を反映するために、「systemctl restart NetworkManager」を実行し、enp0s3にIPアドレスが割り当てられているか確認してください。

    systemctl restart NetworkManager
    ip address show enp0s3

  7. nmcliコマンドを利用して、接続一覧を確認してください。

    nmcli connection show
    nmcli c show

  8. nmcliコマンドを利用して、ネットワークを無効にしてください。

    nmcli network off
    nmcli connection show または nmcli c show

  9. nmcliコマンドを利用して、ネットワークを有効にしてください。

    nmcli network on
    nmcli connection show または nmcli c show

  10. nmcliコマンドを利用して、enp0s3インターフェイスを無効にしてください。

    nmcli connection down enp0s3 または nmcli c down enp0s3
    nmcli connection show または nmcli c show

  11. nmcliコマンドを利用して、enp0s3インターフェイスを有効にしてください。

    nmcli connection up enp0s3 または nmcli c up enp0s3
    nmcli connection show または nmcli c show

  12. studentユーザに戻りましょう。

    exit

2.nmcliコマンド～ホスト名編～

  1. カレントディレクトリをホームディレクトリに変更してください

    cd
    pwd

  2. rootユーザに変更してください。

    su -

  3. nmcliコマンドを利用して、ホスト名を「tokyo-ec.com」に変更してください。

    nmcli general hostname tokyo-ec.com または nmcli g hostname tokyo-ec.com
    hostnamectl status

  4. systemctlコマンドを利用してsystemd-hostnamedをrestartしてください。

    systemctl restart systemd-hostnamed

  5. hostnamectlコマンドを利用して、ホスト名情報を確認してください。

    hostnamectl status

  6. catコマンドで/etc/hostnameファイルを確認してください。

    cat /etc/hostname

  7. studentユーザに戻りましょう。

    exit

3.nmtuiコマンド

  1. カレントディレクトリをホームディレクトリに変更してください

    cd
    pwd

  2. rootユーザに変更してください。

    su -

  3. nmtuiコマンドを実行して、[接続の編集]-[enp0s3]-[取り消し]-[back]-[終了]を選択して、一連の流れを確認してください。

    nmtui
    [接続の編集]-[enp0s3]-[取り消し]-[back]-[終了]

  4. studentユーザに戻りましょう。

    exit
