FluentdからNginxのアクセスログ確認

FluentdからNginxのアクセスログ確認手順

  1. FluentdのコンテナーでNginxのログを確認
    docker-compose exec -T fluentd tail /var/log/nginx/docker.access.log
  2. Fluentdのコンテナーにexec
    docker-compose exec fluentd bash
  3. Fluentdのconfファイルから設定を確認
    cat /fluentd/etc/fluent.conf
  4. execを終了
    exit
