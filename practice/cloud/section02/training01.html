Apache Web Serverのインストール

Apache Web Serverのインストール手順

  1. httpdのインストール
    yum install -y httpd
  2. httpdのバージョン確認
    httpd -v
  3. httpdの自動起動設定
    systemctl enable httpd
    systemctl list-unit-files --type=service | grep httpd
  4. httpdの起動
    systemctl start httpd
    systemctl status httpd
