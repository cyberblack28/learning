ansible-host作業

SSH設定変更手順

  1. 元ファイルのバックアップ 
    # cp -p /etc/ssh/sshd_config /etc/ssh/sshd_config.org
  2. sshd_configの設定変更
    # vim /etc/ssh/sshd_config

---------------------以下赤字の内容に変更---------------------------
・
・省略
・
PermitRootLogin yes //←「#PermitRootLogin no」変更⇒「PermitRootLogin yes」
PasswordAuthentication yes //←「PasswordAuthentication no」変更⇒「PasswordAuthentication yes」
PasswordAuthentication yes //←「PasswordAuthentication no」変更⇒「PasswordAuthentication yes」
・
・省略
・
---------------------[Esc + :wq]で保存終了します。-------------------------
  3. sshdの再起動 
    # systemctl sshd restart
